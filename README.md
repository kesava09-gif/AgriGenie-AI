An AI-powered smart farming platform that helps farmers make better agricultural decisions using Artificial Intelligence.

- 🌱 Crop Recommendation
- 🤖 AI Farmer Assistant (Google Gemini)

- Google Gemini API
- Pandas
- NumPy
